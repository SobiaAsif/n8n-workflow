## 📧 Email Newsletter – Daily Tech News

This workflow automatically sends a daily technology 
news email to my Gmail every morning at 9:00 AM.

### ✨ What It Does:
- Fetches the latest technology news automatically
- Sends a formatted email newsletter to Gmail
- Runs every day at 9:00 AM automatically

### 🛠️ Tools & Services Used:
- **n8n** – Workflow automation
- **Gmail** – Email delivery
- **RSS Feed / News API** – News source

### ⚙️ How to Use:
1. Import the `email-newsletter.json` file into n8n
2. Add your own Gmail credentials
3. Set your preferred email address
4. Activate the workflow

### 📌 Note:
Credentials are not included in the workflow file.
You will need to add your own API keys and Gmail 
credentials after importing.
